# flask_bookstore
# 图书商城系统-网上书店

# 网站介绍  
游客可以浏览、搜索、查看图书，查看评论等  
登录用户可以 浏览 搜索 查看 图书，查看 发布 回复 评论，添加到购物车，收藏，添加收货信息，购买 使用支付宝支付、订单操作 等  
管理员 可以 管理 图书、订单、滚屏图片 等。
